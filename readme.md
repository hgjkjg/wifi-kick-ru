Описание

Wifi-kick — инструмент для тестирования безопасности Wi-Fi сетей. Позволяет проводить деаутентификацию клиентов, глушение диапазонов и определять производителей устройств по MAC-адресам с использованием локальной OUI-базы.

**Только для тестирования собственных сетей!**

Возможности:

- **Деаутентификация всех клиентов** сети
- **Глушение всего диапазона** (2.4 ГГц, 5 ГГц или оба одновременно)
- **Прицельная атака** на конкретное устройство
- **OSINT-сканирование** с определением производителя (Apple, Samsung, Xiaomi и др.)
- **Локальная OUI-база** для мгновенного определения без интернета
- **Автоматическое определение Wi-Fi адаптера**
- **Безопасный возврат адаптера** в нормальный режим после атаки

Установка:

git clone https://github.com/Punisher-ULTRA/wifi-kick-ru.git
cd wifi-kick-ru
chmod +x wifi-kick.sh

Зависимости:

sudo apt install mdk4 aircrack-ng -y

Исппользование:

sudo ./wifi-kick.sh
или 
sudo bash wifi-kick.sh

Далее следуй интерактивному меню.

Режимы

1 - Выбить ВСЕХ из сети

2 - Глушение всего диапазона

3 - Прицельная атака на устройство

4 - Только сканирование с определением производителя (OSINT)

Диапазоны

Режим	     Каналы
1	     Только 2.4 ГГц (1-14)
2	     Только 5 ГГц (36-165)
3	     ВСЁ сразу (2.4 + 5 ГГц)

Как это работает

 Скрипт переводит Wi-Fi адаптер в режим мониторинга
 Сканирует эфир, собирая MAC-адреса устройств
 Определяет производителя каждого устройства через OUI-базу
 Отправляет пакеты деаутентификации выбранной цели
 Возвращает адаптер в нормальный режим

OUI-база

Для мгновенного определения производителей (без интернета) положи файл oui.csv в папку со скриптом. Формат:

Registry,Assignment,Organization Name,Organization Address

MA-L,286FB9,Nokia Shanghai Bell Co., Ltd.,...

MA-L,F4EAB5,Extreme Networks Headquarters,...

Еще одно предупреждение

Данный инструмент предназначен исключительно для:

   Тестирования собственных Wi-Fi сетей

   Образовательных целей

   Пентестов с письменного разрешения владельца сети

Атака на чужие сети без разрешения незаконна и преследуется по закону.
